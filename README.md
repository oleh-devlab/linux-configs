# Linux Configurations

This repository serves as a central collection for my personal Linux configuration files (dotfiles). It is intended to house various configurations for my system to keep everything organized, backed up, and easily deployable.

## Current Configurations

Currently, the repository contains the following:

### [Waybar](./waybar)
A highly customizable Wayland bar for Sway and Wlroots based compositors.
- **`config.jsonc`**: Defines the layout of the bar and the behavior of both standard and custom modules (such as battery energy, system temperature, and SSD stats).
- **`style.css`**: The styling file, utilizing a clean, pill-shaped design with custom color accents for different states.

## Future Plans

This repository will expand over time. I plan to add more configuration files here in the future (e.g., for Sway, terminal emulators, editors, and other system utilities) as I continue to organize my setup.
